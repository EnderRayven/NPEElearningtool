# 默认题库工作区

首次启动后点击网页顶部的“文件夹”，选择当前 `题库工作区` 目录。

目录中已经预建两个默认题库：

- `高等数学 · 基础篇`
- `线性代数 · 强化篇`

将题目图片和答案图片放入对应的普通文件夹，然后在网页中点击“已连接”重新扫描。

```text
Q-01-1-01.png       第 1 章、第 1 小节、第 1 题的题目图
Q-01-1-01.1.png     同一道题由多张图片组成时的第 1 张
Q-01-1-01.2.png     同一道题由多张图片组成时的第 2 张
A-01-1-01.png       第 1 章、第 1 小节、第 1 题的答案图
```

网页连接后会自动生成 `题库数据.json`。题库重命名、学习标注和题库结构修改都会实时写入该文件。
