# 用户数据

应用会在此目录生成 `用户数据.json`，用于保存熟练、模糊、错题等个人学习标记、每日做题记录、学习轮次、题目文字/手写笔记和考试日期等用户设置。

学习记录按轮次相互隔离。升级后原有记录会自动归入第 1 轮，应用默认预设 5 轮；可在顶部“设置 → 学习轮次”中切换或新增，全部轮次都会随完整备份保存。

旧版状态、每日记录和单独设置会在首次成功读取后统一写入当前格式，并清理重复旧字段；未产生记录的预设轮次不会写入空数据。

题目笔记保存在顶层 `notes` 字段中，以题目 ID 关联，不随学习轮次切换。文字笔记直接保存文本；手写笔记保存可继续编辑的矢量笔迹，而不是不可修改的截图。浏览器端使用 IndexedDB 保存笔记，连接工作区或导出完整备份时会同步写入结构化用户数据。

该文件与 `默认题库/题库数据.json` 分开维护，并已加入 `.gitignore`，不会随题库项目数据提交到版本库。删除该文件会清空磁盘上的个人记录与设置，但不会影响题库、题图或答案图。
